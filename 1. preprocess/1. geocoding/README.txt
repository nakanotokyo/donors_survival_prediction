README

1. data contains SQL to generate file that will be input for donors and non-profits (ONG) geolocation 

2. donors_geolocation and 3. ong_geolocation contains code to geolocate donors and non-profit using OpenStreetMap

4. distance contains code to calculate distance between donors and non-profits. the output is input in the dataset for the model

The process must be run step by step from 1 to 4
