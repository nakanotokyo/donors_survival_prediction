README

1. Data contains SQL to generate file that contains all the occupation filled in the form. 

2. Dataset from step 1 is preprocessed and Levenshtein distance is computed to normalize occupations. 

3. Results contains the dataset uploaded en step 1 plus the normalized occupation generated in step 2. It also contains a graph of occupation distribution. 

The process must be run step by step from 1 to 3.
